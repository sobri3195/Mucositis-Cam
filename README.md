# Mucositis-Cam

**Judul studi:** *Mucositis-Cam: Prediksi Mukositis Berat pada RT Kepala-Leher dari Foto Intraoral Harian dan Dose Map*

## Ringkasan desain penelitian
- **Desain:** Kohort prospektif prediktif
- **Populasi (P):** Pasien radioterapi (RT) kepala-leher
- **Intervensi/Indeks (I):** Model AI multimodal berbasis foto intraoral smartphone harian + distribusi dosis (dose map)
- **Komparator (C):** Penilaian klinis rutin tanpa model AI
- **Outcome (O):**
  - **Primer:** Mukositis grade ≥3
  - **Sekunder:** Kebutuhan feeding tube, interupsi RT, dan kualitas hidup (QoL)

## Rumusan PICO
| Komponen | Deskripsi |
|---|---|
| **P (Population)** | Pasien RT kepala-leher |
| **I (Intervention/Index)** | AI multimodal (foto intraoral smartphone + distribusi dosis) |
| **C (Comparator)** | Penilaian klinis rutin tanpa model AI |
| **O (Outcomes)** | Mukositis grade ≥3 (primer), feeding tube, interupsi RT, QoL |

## Tujuan
Mengembangkan dan memvalidasi model prediksi risiko mukositis berat secara dini selama RT kepala-leher menggunakan data multimodal harian, untuk mendukung pengambilan keputusan klinis yang lebih cepat dan terpersonalisasi.
