# Mucositis-Cam

Prototype antarmuka web untuk studi **Mucositis-Cam** dengan fokus tampilan **mobile responsive** dan **bottom navigation**.

## Fitur UI
- Layout mobile-first dengan konten berbasis kartu.
- Bottom navigation (Beranda, Pasien, Analitik, Profil).
- Navigasi mengaktifkan section konten tanpa reload halaman.
- Tetap responsif pada layar tablet/desktop.

## Menjalankan
Cukup buka file `index.html` di browser.
